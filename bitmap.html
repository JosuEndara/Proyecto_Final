Información importante

MARS: Si no se dispone de una computadora física con un procesador MIPS, se puede probar la implementación en un entorno simulado dentro de MARS.

BITMAP DISPLAY: Para la simulación de la interfaz gráfica se utiliza la herramienta "Pantalla de mapa de bits" (Bitmap Display en inglés). Esta herramienta permite simular una pantalla de visualización gráfica como una matriz de "unidades" almacenadas en memoria. Cada unidad consta de un grupo de píxeles definido en la configuración como "ancho/alto de la unidad en píxeles". El tamaño de la matriz es el número total de unidades en la pantalla. Además, Bitmap Display permite ajustar la configuración de visualización, como el número de bits por píxel, la dirección base de la pantalla y la ordenación de bytes.

KEYBOARD AND DISPLAY MMIO SIMULATOR: Para simular la entrada de teclado se utiliza la herramienta "Teclado y pantalla MMIO Simulator" (Keyboard and Display MMIO Simulation en inglés). Permite simular la entrada de teclado y la visualización de salida en la pantalla. El teclado simulado se puede configurar para enviar eventos de teclado en una dirección de memoria específica.
